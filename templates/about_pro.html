About The Project

A sign-language recognition project built with computer vision, machine learning, and a more polished web interface.

This system turns hand gestures into readable text by extracting landmarks, classifying the gesture, and presenting the result through image and live-camera workflows.

Flask app MediaPipe landmarks scikit-learn model
What it does

From hand shape to text output

The app detects a hand, extracts landmark-based features, and sends them through the trained model to predict which sign is being shown. The result is displayed with confidence and alternative matches so the output feels more transparent.

How you use it

Two practical modes

  • Upload a single image to analyze one clear hand sign.
  • Use the live studio to run predictions continuously from the webcam.
  • Review the detected label, confidence, and ranked alternatives.
