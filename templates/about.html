About the project

This is a sign language translator that turns hand gestures into text. You can upload a photo of a sign or use the live webcam: the app finds your hand, reads its shape with computer vision, and runs a machine learning model to predict which sign it is. Results include the label, confidence, and optional on-screen box around the hand.

You train the model on your own images per sign class (folders under data/raw), then extract landmarks and train so the app learns your vocabulary.

What we used

  • Python — core language
  • Flask — web server and pages
  • OpenCV — webcam and image handling
  • MediaPipe Hands — 21 hand landmarks per frame
  • NumPy — feature vectors from landmarks
  • scikit-learn — StandardScaler + HistGradientBoostingClassifier (pipeline saved as models/model_bundle.pkl)
  • Joblib — saving and loading the model
  • Pillow — decoding images from the live webcam API
  • HTML, CSS, JavaScript — UI, live camera, overlays
